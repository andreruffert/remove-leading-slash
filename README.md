> Remove leading slashes from a given string

## Install

```console
$ npm i remove-leading-slash
```

## Usage

```js
import removeLeadingSlash from 'remove-leading-slash';

removeLeadingSlash('/some/random/path/');
// => 'some/random/path/'
```
